Create a makerom folder on desktop. C:\User\YourNameHere\Desktop\makerom 
Move makerom-x86_64.exe and the batch files to that folder
Rename your game to ROM.cia and move to makerom folder
Edit RUN First.bat to your name. (i.e. C:\User\--> YourNameHere <--\Desktop\makerom)
Run RUN First.bat and you are good!
